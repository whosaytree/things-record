# 更新日志

## 1.0.0 - 2026-04-21

首个可用版本。

### 新增

- 建立手机优先的个人物品记录网站。
- 支持按类别展示物品。
- 支持新增、编辑、删除物品。
- 支持物品名称、类别、评分、描述、图片和购买链接。
- 支持物品详情页。
- 支持类别新增、重命名和删除。
- 支持删除类别时迁移该类别下的物品。
- 支持关键词检索，匹配物品名称和描述。
- 支持 JSON 数据导出和导入。
- 支持默认类别去重，避免开发模式下重复初始化。
- 支持基础 PWA 配置和主屏安装。

### 说明

- 当前版本为本地优先应用，不需要服务器。
- 数据保存在浏览器 IndexedDB 中。
- 多设备同步和云备份不包含在 1.0.0 中。
