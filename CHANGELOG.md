# 更新日志

## 2.0.0 - 2026-04-21

### 新增

- 支持将生产构建产物部署到静态网站后在手机端直接使用。
- 新增 GitHub Pages 自动部署工作流，推送到 `main` 后可自动构建并发布 `dist/`。
- README 增加 GitHub Pages、Netlify、Vercel 和 Cloudflare Pages 的部署说明。

### 改进

- Vite 生产构建改为相对资源路径，支持根域名和子路径部署。
- PWA manifest 的启动地址、作用域和图标路径改为相对路径。
- Service worker 注册地址和离线缓存路径改为按部署地址解析。
- Service worker 缓存版本更新为 `v2.0.0`，避免手机继续使用旧缓存。

## 1.0.1 - 2026-04-21

### 改进

- 调整整体视觉风格，弱化大面积渐变、玻璃感和过大的圆角。
- 统一主题色为 `#88C9A0`，并补齐浅色、深色、边框和聚焦状态。
- 优化首页分类面板、物品卡片、搜索框、表单、详情页和设置页的层级与间距。
- 将“记录原则”提示条改为更轻量的主题色样式。
- 移除页头“手机优先记录”文案。
- 移除底部导航中的“新增”入口，保留首页和设置两项。
- 将底部导航改为浅色面板，降低视觉突兀感。

## 1.0.0 - 2026-04-21

首个可用版本。

### 新增

- 建立手机优先的个人物品记录网站。
- 支持按类别展示物品。
- 支持新增、编辑、删除物品。
- 支持物品名称、类别、评分、描述、图片和购买链接。
- 支持物品详情页。
- 支持类别新增、重命名和删除。
- 支持删除类别时迁移该类别下的物品。
- 支持关键词检索，匹配物品名称和描述。
- 支持 JSON 数据导出和导入。
- 支持默认类别去重，避免开发模式下重复初始化。
- 支持基础 PWA 配置和主屏安装。

### 说明

- 当前版本为本地优先应用，不需要服务器。
- 数据保存在浏览器 IndexedDB 中。
- 多设备同步和云备份不包含在 1.0.0 中。
